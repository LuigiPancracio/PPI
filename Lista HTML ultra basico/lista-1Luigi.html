R: É como uma biblioteca global, onde livros, artigos, vídeos e outros conteúdos se juntam. Essa biblioteca é a Web, um universo de informações acessível através de softwares como Chrome, Firefox e Safari. Para navegar nesse universo, você precisa de uma estrada digital, uma rede de computadores interligados que se estende por todo o planeta. B. O que é arquitetura cliente-servidor utilizada na web ? Explique o carregamento de uma página passo a passo

R: A arquitetura cliente-servidor na web refere-se à distribuição de funções entre um cliente e um servidor. O cliente solicita recursos, e o servidor fornece esses recursos. O carregamento de uma página web se baseia nos seguintes passos: O (cliente) envia uma solicitação HTTP para o servidor, depois o servidor processa a solicitação e responde de volta com os arquivos necessários, como HTML, CSS e JavaScript, assim o navegador interpreta o HTML para construir e estruturar a página solicitando possiveis recursos adicionais, como imagens, scripts e estilo de letras, por fim renderizando a página,e exibindo-a para o usuário.
